🌱 Happy Herbivore

Happy Herbivore is a digital ordering system designed for a vegan restaurant. The application allows customers to browse meals, view nutritional information, and place orders through a simple and user-friendly interface.

The goal of this project is to improve the customer ordering experience through clear navigation, helpful feedback, and a clean visual design.

📋 Features
🥗 Menu

Separate Lunch and Dinner sections

Clear vegan tags for vegan products

Calorie count displayed for menu items

🛒 Ordering System

Add items to an order

View your current order

Cancel order option

Prevent checkout when the order is empty

⚠️ Pop-ups & Notifications

Pop-up when trying to pay without selecting items

Idle detection pop-up (“Are you still there?”)

🧾 Order Completion

Digital receipt after completing an order

Screen reset so the next customer can start a new order

🎨 User Interface

Improved styling for Lunch and Dinner menus

Styled buttons for better usability

Slideshow background on the home screen

Larger navigation bar logo

Improved order screen layout

Clearer vegan labels

 Purpose

This project was created to practice building an interactive ordering interface while focusing on:

User experience (UX)

Interface design (UI)

JavaScript functionality

Error handling through pop-ups
